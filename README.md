# URL Shortener
A full-stack URL shortening application built with Spring Boot, providing a REST API and a web user interface for managing shortened links.The system generates unique 10-character identifiers for URLs, supports automatic redirection, and tracks visit statistics.It includes advanced validation for user inputs (name, URL, password) with detailed, user-friendly error messages.
The project features a Spring MVC + Thymeleaf frontend with Bootstrap. The application supports messages internationalization with languages (English, Polish, German).
## Technologies & Techniques
- Java
- Spring Boot
- Spring Web 
- Spring MVC
- Thymeleaf
- Bootstrap
- H2 Database
- Spring Data JPA / Hibernate (ORM)
- Bean Validation
